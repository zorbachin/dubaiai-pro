# Naval framework set (captured from IG, 2026-06-12)

Four reusable audit prompts — run any of them with `the nucleus + 30-day audit
as the ABOUT-ME inputs`. Executed for Zorba: see `naval-execution.md`.

1. **Specific Knowledge Excavator** — cross-reference obsessions × career
   detours × undervalued skills → niche nobody else holds; reject anything
   teachable in a bootcamp; 3 business models scored market/(6−competition)/
   leverage-multiplier; never labor-based.
2. **Leverage Stack Auditor** — map income/activities to labor/capital/code/
   media; leverage score 1–5 each; Leverage Index = Σ(score×rev%)÷100; find
   the biggest leak; 3 concrete labor→code/media conversions in 30 days.
   Rules: hourly = 1 regardless of rate; flag income that dies if you stop 6mo.
3. **Productize Yourself Blueprint** — the transformation you deliver → 3
   product formats scored leverage/feasibility(30 days)/margin(>70%); must
   have a named proprietary method; reject anything needing live presence.
4. **Time-for-Money Leak Detector** — every activity = Time Rented vs Equity
   Building; equity only counts if income survives you stopping 6 months;
   rank conversions; project 24-mo equity gap at 5% monthly compound.
