# 📅 Posting Cadence & Strategy — "The AI Guy" / LetAIDoIt

_The point: hit every platform consistently without burning out. Create once,
distribute all week. Published = done._

---

## 🎯 The principle: BATCH, don't scramble

You don't post daily by *creating* daily — that's the trap that kills consistency.
You **create once a week**, and the system distributes it all week. ADHD-proof:
one creation block, then near-automatic distribution.

The pipeline that makes this work (proven 2026-06-02):
**Record → drop in Drive "buffer content" folder → Claude atomizes + queues all
platforms in Buffer → you approve.** The only manual post is Facebook (personal
profile — Meta blocks automation).

---

## 🗓️ The weekly rhythm

### Sunday — BATCH (1 block, ~60–90 min)
- Record **1 pillar video** (the park works — corny is the brand).
- Brain-dump the week's angle (voice note is fine).
- Drop the video in the Drive **"buffer content"** folder.
- Claude atomizes the pillar → a week of posts (clips, quote cards, carousel,
  written posts, meme, newsletter) and **pre-loads Buffer**.

### Mon–Fri — DISTRIBUTE (5 min/day, mostly approving)
- Daily standup surfaces **today's one post** — pre-decided, video/caption ready.
- You: post Facebook by hand (2 min) + approve the Buffer queue. Done.

### Sunday — NEWSLETTER
- The week's pillar becomes the newsletter (links to the video). Send.

## 📈 Cadence (the floor — scale up when batched, never below this)

| Platform | Frequency | How |
|---|---|---|
| **Facebook** (personal, #1) | 3–5×/week | Manual (your warm audience — worth the 2 min) |
| **Instagram** (zorb_ai) | 4–5×/week | Buffer auto |
| **LinkedIn** | 3×/week | Buffer auto |
| **X** | 5×/week | Buffer auto (lowest effort, repurpose quotes) |
| **TikTok** | 3×/week | Connect to Buffer, then auto |
| **Newsletter** | 1×/week (Sun) | Gmail/email tool |

**Floor = 3 posts/week minimum, even on a bad week.** Consistency > volume.

---

## 🛡️ Sustainability guardrails (so we actually hit them)

1. **One batch beats seven scrambles.** Protect the Sunday block.
2. **Corny/raw is fine — published beats polished.** Don't let editing stall a post.
3. **Facebook is the only manual one.** Everything else is queue-and-approve.
4. **The standup is the enforcer** — every morning it names the day's post so
   nothing slips silently. Misses get named, not buried.
5. **The flywheel:** 1 video → 6+ pieces. You're never starting from a blank page.

## 🔁 The repeatable command (next video)
1. You drop the clip in Drive "buffer content" + say the word.
2. Claude grabs it, atomizes the message per platform, queues all of Buffer
   with the video attached (Drive `usercontent.google.com/download` URL).
